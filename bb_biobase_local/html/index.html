Biobase

Open this page from any machine on the same LAN. Name resolution should use Use bb_biobase_local on the host (mDNS in mdns/ or /etc/hosts — see README). If the page URL includes a port (default :8880), use that in Grafana’s GF_SERVER_ROOT_URL too. Links below are path-relative. Monitoring, logs, and game control:

  • Grafana Dashboards, Postgres panels, and alerts.
    /bb/
  • Loki Log aggregation API and UI (container logs via Promtail).
    /loki/
  • Prometheus Metrics UI (cAdvisor, Rcon exporter) — on host port 19090.
  • CS2 control Bot start/stop and status (RCON). Same as direct port 8765.
    /cs2/
Set GF_SERVER_ROOT_URL to this origin plus /bb/ (e.g. http://biobase.local:8880/bb/ with the default gateway port) so redirects and cookies line up. Direct access to old ports (3003, 19090, 8765) is unchanged.
